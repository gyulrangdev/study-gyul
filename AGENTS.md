# Repository Guidance

When helping with Deep-ML problems in this repository:

- Do not reveal the final answer unless the user explicitly changes that rule.
- Teach required syntax and reasoning first.
- Build intuition with NumPy before moving to PyTorch.
- Save each problem under `ai/deep-ml/problems/<number>-<slug>/`.
- After a problem is solved by the user, add the final solution to that problem folder and publish it to the remote repository.
- Keep solution files small and focused. Avoid new dependencies.
