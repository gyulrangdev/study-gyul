# 멀티패러다임 프로그래밍 스터디

## 📚 스터디 소개

이 스터디는 『멀티패러다임 프로그래밍』(저자: 마플코퍼레이션, 한빛미디어 출판)을 함께 읽고, 객체지향, 함수형, 명령형 등 다양한 프로그래밍 패러다임을 실습과 토론을 통해 깊이 있게 이해하는 것을 목표로 합니다.

## 🗓️ 진행 방식

- 매주 한 챕터씩 읽고, 주요 개념 및 예제 코드를 함께 리뷰합니다.
- 각자 맡은 부분을 정리하여 발표하고, 질의응답 및 토론을 진행합니다.

## 📖 책 목차

1. 멀티패러다임이 현대 언어를 확장하는 방법
2. 함수형 프로그래밍과 타입 시스템, 그리고 LISP
3. 코드:객체:함수 = Generator:Iterator:LISP = IP:OOP:FP
4. 비동기 프로그래밍
5. 실전 함수형 프로그래밍
6. 멀티패러다임 프로그래밍
7. 객체 지향 프론트엔드 개발, 그리고 멀티패러다임적 접근과 응용
