# Deep-ML Practice

This folder mirrors Deep-ML practice work.

## Flow

1. Read the problem statement.
2. Restate the shape transformation or algorithm in plain language.
3. Practice with NumPy to check dimensions and indexing.
4. Rebuild the same idea in PyTorch.
5. Only then write the final submitted solution.
6. Publish changes with `../scripts/publish_deep_ml.sh`.
