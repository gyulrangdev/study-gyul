# Member Notes

각자 이름 또는 닉네임으로 폴더를 만들고 주차별 정리를 남깁니다.

예시:

```text
members/
└── gyuri/
    ├── week-01.md
    └── week-02.md
```

권장 파일 구성:

- `week-01.md`: 1주차 개인 정리
- `week-02.md`: 2주차 개인 정리
- `questions.md`: 스터디 중 계속 가져갈 질문
- `experiments/`: 직접 실행한 코드나 노트북

개인 정리는 완성된 글이 아니어도 괜찮습니다. 헷갈린 점과 질문을 남기는 것만으로도 충분합니다.
