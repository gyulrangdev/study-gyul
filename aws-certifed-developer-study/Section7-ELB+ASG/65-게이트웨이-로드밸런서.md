# 주요 내용 요약

게이트웨이 로드 밸런서(GWLB)의 특징과 작동 원리

# GWLB 개요

## 주요 용도

- 타사 네트워크 가상 어플라이언스 관리
- 네트워크 트래픽 검사
- 방화벽 구현
- 침입 탐지 및 방지 시스템(IDPS)
- 심층 패킷 분석

## 기술적 특징

- Layer 3 (네트워크 계층) 작동
- GENEVE 프로토콜 사용 (포트 6081)
- VPC 라우팅 테이블 자동 업데이트

# GWLB 주요 기능

1. 투명 네트워크 게이트웨이

   - VPC 트래픽의 단일 진입/출구 지점
   - 모든 트래픽 검사 가능

2. 로드 밸런서
   - 가상 어플라이언스 플릿 간 트래픽 분산
   - 트래픽 분석 및 처리

# 트래픽 흐름

```
사용자 → GWLB → 가상 어플라이언스(검사) → GWLB → 애플리케이션
```

# 대상 그룹 옵션

1. EC2 인스턴스

   - 인스턴스 ID로 등록

2. IP 주소
   - 프라이빗 IP만 지원
   - 온프레미스 가상 어플라이언스 연결 가능

# 시험 관련 중요 포인트

1. GWLB 사용 사례

   - 보안 검사
   - 트래픽 분석
   - 방화벽 구현

2. 기술적 특징

   - Layer 3 동작
   - GENEVE 프로토콜
   - 포트 6081

3. 아키텍처 이해
   - 트래픽 흐름
   - 가상 어플라이언스와의 통합
